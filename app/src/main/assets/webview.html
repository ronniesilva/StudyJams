WebView

Uma WebView é simplesmente uma View que exibe página web.

Ele usa o motor de renderização WebKit para exibir páginas da web e inclui métodos para navegar para frente e para trás através de uma história, zoom in e out, realizar pesquisas de texto e muito mais.

Esta classe pode exibir conteúdo on-line dentro de sua activity.

WebView webView = (WebView) findViewById(R.id.webView);
webview.loadUrl("http://www.meetup.com/pt-BR/GDG-Salvador/");

Também é possível carregar uma página html local.

webview.loadUrl("http://www.meetup.com/pt-BR/GDG-Salvador/");

Por padrão o uso de JavaScript é desabilitado e o erros serão ignorados, porém você pode habilitá-lo.

webView.getSettings().setJavaScriptEnabled(true);

Com o uso do WebView permitirá realizar formatação CSS.

<style> p{ font-family: tahoma, arial; font-size: 12px; text-align: justify; } </style>
