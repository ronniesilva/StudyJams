Neste app vamos mostrar algumas formas de exibição de texto dentro de uma activity.

A primeira forma é o componente TextView.

A segunda é inserindo código html como parâmetro diretamente no Textview

A terceira é colocar o código html dentro da value "string.xml". No entanto, esta abordagem tem diversas limitações, e você provavelmente vai querer mostrar o seu HTML em um WebView

Logo, a quarta é apresentar o html dentro do componente webview.

Obs: Este App possui cunho didático é deve ser analisado juntamente com seu código-fonte.
